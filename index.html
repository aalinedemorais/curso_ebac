Aula 4: HTML, CSS e DOM

HTML

  • Significa Hyper Text Markup Language
  • É a linguagem usada para marcação padrão para criação de páginas da web
  • Os elementos HTML informam ao navegador como exibir o conteúdo.

    • CSS

  • Significa Cascading Styles Sheets
  • Aplica-se o estilo a um elemento pai e aplicará também aos filhos
  • Serve para controlar cor, fonte, tamanho do texto, espaçamento entre elementos, etc

    • DOM

  • Significa Document Object Model
  • Quando uma página web é carregada, o navegador cria um DOM da página
  • Foi criado pela W3C com objetivo de desenvolver um padrão para linguagens de script para navegadores

    • Para mais informações acessar W3C
